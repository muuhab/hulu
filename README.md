# Hulu CLone

## Features

-Movies information
-Show movies by genre
-Show trendy movies

## Tech

- [NextJS](https://nextjs.org/)
- [Tailwind-CSS](https://tailwindcss.com/) 


## Getting Started
First, install the packages
```bash
npm i
```
Second, run the development server:

```bash
npm run dev
# or
yarn dev
```
